🚀 DigiTools – Premium Digital Tools Marketplace

📌 About the Project

DigiTools is a modern and visually stunning web application where users can explore and purchase premium digital tools to boost productivity and creativity.
It features a clean UI, smooth user experience, and a simple cart system for managing purchases.

🛠️ Tech Stack
React.js
Tailwind CSS
JavaScript (ES6+)
React Toastify
HTML5

✨ Features:
1. Dynamic Product Listing
2. Interactive Cart System
3. Tab-Based Navigation
